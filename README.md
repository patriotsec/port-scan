# Network Scanning Fast
## Install :
### sudo apt-get update && sudo apt-get upgrade
### sudo apt-get install python3
### pip3 install argparse
### pip3 install colorama
### pip3 install thread
### git clone https://github.com/patriotsec/port-scan.git
### cd portscan
### python3 main.py -h
